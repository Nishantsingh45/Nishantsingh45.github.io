Offline AI Dictation Tool

Native-Rust push-to-talk dictation for Windows powered by Whisper.cpp. Fully offline — no cloud, no API key, no GPU required. CPU, Vulkan, and CUDA backends supported on any Windows 10/11 machine.

Built because every existing option was either cloud-dependent or bloated. Ended up orders of magnitude leaner than the Electron alternatives.
